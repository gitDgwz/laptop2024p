为什么我要你在线上买而不在线下买?

线上

  1. 如今电脑配置和价格在网上十分透明,稍微花一些时间便可了解大概,避免购买到高价低配的电脑.

  2. 线上只要在官方店购买,不买杂牌子最多买贵一点.

  3. 一般可以七天退

线下

  1. 电脑小白去电脑店非常容易被忽悠,在线下购买电脑被宰的事件更是数不胜数.

  2. 线下购买,可以没有底线的黑,三千的卖五千,搞不好还是翻新机(老机器翻新)

  3. 买了发现问题想退很难

一些黑商在你问其配置的时候往往只会说一些无关紧要的参数, 比如显卡,他说8G(8G指的是显存,是个次要参数,3050 8G和3070 8G,差得不是一点半点), CPU他说i7(某些老i7性能比i5低,)

一台电脑里最重要的一般为CPU和显卡.最直接能指明其好坏的就是具体型号,而不在于一些次要参数

显卡型号 如RTX 4060 Ti 8G
CPU型号 如 i9-9900

Note

CPU和显卡具体还分哪个厂商等一些细致参数,新手可以暂时不用考虑
